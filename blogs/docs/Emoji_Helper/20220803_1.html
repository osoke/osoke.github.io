    # 简介

    这是一款输入 emoji 表情的小工具, 仅支持 Windows 平台。

    由于平时经常使用 emoji 表情, 但是 win10 自带的 emoji 工具有一个缺点, 无法通过关键词查找, 你想找某个 emoji 表情, 得把所有表情符号挨个翻一遍, 翻找的过程中很容易漏掉要查找的目标表情, 这会导致翻了一遍还是找不到, 进而导致烦躁, 打断思路...对于经常使用 emoji 的人来说真是太难了!

    所以我开发了这款小工具, 支持中-英文关键词查找, 想要什么直接输入, 有了它, emoji 便可随手拈来。

    # 使用方法

    运行程序

    • 呼出面板: 双击-左Ctrl (可以改)
    • 搜索 emoji: 输入「关键词」并回车
    • 选择 emoji: 鼠标点击想要的 emoji, 自动复制到剪贴板并粘贴到输入窗口。
    • 拖动窗口: 拖动上方输入框
    • 隐藏面板: 点击 Esc 键

    # 搜索模式

    用中文搜索的话, 两种模式差别不大, 直接搜即可。

    # 模糊搜索(默认)

    比如: 搜索 dog 会匹配任何包含 d o g 这三个字母的条目, 会包含冗余结果 (如 drooling), 想要获取精确的结果, 请使用「精确搜索」

    # 精确搜索

    语法: ;关键词

    还是以 dog 为例, 如果确定搜的是「小狗」而非别的内容, 在前面添加「中·英文分号」;dog, 就会精确匹配到包含 dog 的条目

    # 操作系统 & emoji 版本

    操作系统对 Unicode emoji 的支持是相对滞后的 (Unicode 每年都在更新), 如果你的电脑系统不是 win10, 部分 emoji 可能无法正常显示。

    虽然 Emoji Version 14.0 在 2021-9-14 早已发布, 但还未得到操作系统的广泛适配。 所以本工具采用目前支持较好的 Emoji Version 13.0
