    # BeeWare 简介

    一个 Python 跨平台 GUI 框架, 可以做到一次编写, 全平台运行。

    2. 安装 Git (opens new window)

    # 2 - 让它变得有趣

    src/helloworld 中 app.py 就是我们的 App 主程序：

    import toga                              # 小部件工具包
from toga.style import Pack              # 样式相关 - 类
from toga.style.pack import COLUMN, ROW  # 样式相关 - 常量

class HelloWorld(toga.App):      # 主窗口
    def startup(self):           # 启动方法
        main_box = toga.Box()    # 定义一个 Box 作为主框 (类似于 div)

        # 定义一个窗口 | 标题 = formal_name = App 名称
        self.main_window = toga.MainWindow(title=self.formal_name)
        self.main_window.content = main_box  # 将 main_box 放入其中(空 Box)
        self.main_window.show()              # 显示窗口

def main():
    return HelloWorld()   # 返回 App 实例

    # 添加一些我们自己的内容

    修改 src/helloworld/app.py 内部的 HelloWorld 类 注意: 别动 HelloWorld 类以外的代码

    class HelloWorld(toga.App):
    def startup(self):
        main_box = toga.Box(style=Pack(direction=COLUMN)) # 定义一个主框
        
        # 标签
        name_label = toga.Label(
            'Your name: ',
            style=Pack(padding=(0, 5))    # 左右 5px 填充
        )
        # 文本输入框 | flex = 灵活 = 吸收其布局轴中的所有可用空间
        self.name_input = toga.TextInput(style=Pack(flex=1))
                    	
        # 定义一个 Box 来保存 上面 2 个小部件
        # ROW 内容水平添加 宽度尽可能窄
        name_box = toga.Box(style=Pack(direction=ROW, padding=5))
        name_box.add(name_label)
        name_box.add(self.name_input)

        # 定义一个 按钮
        button = toga.Button(
            'Say Hello!',
            on_press=self.say_hello,  # 回调
            style=Pack(padding=5)
        )

        # 添加 name_box 和 button 到 main_box (相当于 body)
        main_box.add(name_box)
        main_box.add(button)

        # MainWindow 添加 main_box 并显示
        self.main_window = toga.MainWindow(title=self.formal_name)
        self.main_window.content = main_box
        self.main_window.show()

    # 按钮事件的处理程序
    # 传入 widget
    def say_hello(self, widget):
        # 打印 name_input 输入框的值
        print("Hello, ", self.name_input.value)

    direction=COLUMN

    # 3 - 打包以进行分发

    # 创建 App 脚手架

    第一次打包 App ，需要创建一些配置文件和其他脚手架来支持打包过程。 从 C:\beeware-tutorial\helloworld 目录中，运行：

    briefcase create
    1

    完成此操作后，在项目目录中, 生成与您的平台  (macOS, linux, or windows) 对应的目录，其中包含: 特定于平台的 App 打包配置文件。

    # 构建 App

    现在可以编译 App 为可执行二进制文件

    briefcase build
    1

    # 运行应用

    运行本机 App

    briefcase run
    1

    # 构建安装程序

    briefcase package
    1

    完成此步骤后，该 windows 文件夹将包含一个名为 Hello_World-0.0.1.msi 的安装文件。双击开始安装, 完成后，您的开始菜单中将有一个 Hello World 条目。

    至此, App 已经打包，可以在桌面平台上分发。

    # 5 - 移动端部署

    「部署为 Android App」的过程与「部署为桌面 App」 的过程非常相似。

    # 创建安卓应用并编译

    首先，运行 create 命令。

    briefcase create android
    1

    这将下载一个 Android 应用模板，并将你的 Python 代码添加到其中。 中间需要下载很多东西, 耐心等待。 下载完之后会提示: 接受 Google 的 Android SDK 许可证。一路 y 回车 即可

    完成此操作后，我们现在将在项目中拥有一个 android 目录。其中包含一个名为 Hello World 的 Android 项目。

    编译为 Android APK

    briefcase build android
    1

    由于是第一次 build, 会下载很多 Android SDK 组件, 耐心等待

    移动设备上的 Python 限制（暂时！)

    • 只能使用 「纯 Python」包，即「不包含二进制模块的包」。
    • 这意味着 numpy 、scikit-learn、cryptography 等库不能在移动端使用。

    理论上可以构建一个「使用二进制模块」的移动端 Python App, 但这不是一个简单的任务。如果您希望将此特性添加到 BeeWare 中，请考虑通过成为会员来支持该项目 (opens new window)。

    # 7 - 让界面响应变顺滑

    当 App 发出 Web 请求时，它会等待 API 返回响应，在等待时，界面会无响应。

    # 图形用户界面 event 循环

    当你操作 App 会产生 event, 然后 event 被放入队列中，App 会自动处理 event 队列。App 处理完所有可用 event 后，将重绘界面。

    注意：当 App 处理 event 时, 它无法重绘界面, 如果 event 比较耗时, 此时界面就会「无响应、卡死」。

    此问题「异步」可解。

    # 异步编程

    给函数添加 async 关键字, 声明为「协程」, 表示可以异步调用。 在耗时操作前添加 await 关键字, 表示在等待「耗时操作」返回时, 可以将上下文切换为另一个「协程」

    # 给程序添加异步逻辑

    修改 say_hello() 回调函数:

    async def say_hello(self, widget):         # 1-异步函数
    if self.name_input.value:
        name = self.name_input.value
    else:
        name = 'stranger'
    
    # 2-异步 上下文 ┃ 3-异步 Client
    async with httpx.AsyncClient() as client:   
        # 4-await
        response = await client.get("https://jsonplaceholder.typicode.com/posts/42")

    payload = response.json()

    self.main_window.info_dialog(
        "Hello, {}".format(name),
        payload["body"],
    )
