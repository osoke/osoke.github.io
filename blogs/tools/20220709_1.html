    obsidian 入门

    osoke 2022/7/9 知识管理MarkdownObsidian

    # 是什么

    一款知识管理工具 你可以用它来存放笔记/文章/代码...构建个人知识库/知识体系

    # 特性

    • 开源, 免费
    • 全平台支持
    • 数据存储在本地, 安全
    • 基于 Markdown, 易读易写
    • 持支双向链接, 便于知识之间互相关联
    • 丰富的插件, 极强的扩展能力

    # 下载安装

    • 官网 (opens new window)
    • 官方文档 (opens new window) 有中文

    # 创建库

    主界面

    # 基础操作

    • Ctrl-N 新建笔记
    • Ctrl-O 打开笔记 (没有则新建)
    • Ctrl-W 关闭笔记
    • Ctrl-E 切换编辑/预览模式

    现在你可以自由探索软件的功能, 把界面的按钮都点一下, 看看有什么效果。不用担心弄坏软件, 就把现在这个存储库当作专门测试功能的库, 等搞明白基本操作之后, 你可以再新建一个存储库, 正式开始写内容

    # 外观

    1. 可开启深色模式
    2. 更换主题 (需要魔法)
      • 选一个看着顺眼的, 再微调样式即可
      • 我用的是 Pisum
    3. 设置字体
      • Interface font 应用界面字体
      • Text font 编辑内容字体
      • Monospace font 代码块字体

    # 功能太多, 无法下手怎么办？

    使用工具前, 首先考虑你要做什么, 然后再评估工具合不合适, 不合适就换另一个, 最后的结果应该是你将工具融入自己的工作流

    工具本身的功能再强大, 对你来说不重要, 你要的只是能融入你工作流的那一点点功能, 然后把那一点点功能用好就够了。你要把每个功能都研究一遍？生命是短暂的, 朋友

    正确的做法应该是：就把它当作普通记事本来用(高级一点的记事本), 逐渐探索它的功能, 慢慢的找到最适合自己、同时也是最舒服的使用方式
