    设置 .bat 默认打开方式为 Windows Terminal

    osoke 2022/7/26 CMDbatWindows 文件打开方式

    # 1.修改注册表

    • win+R 输入 regedit 打开注册表
    • 定位到以下 key
    计算机\HKEY_CLASSES_ROOT\batfile\shell\open\command
    1

    双击(默认) ▶ 值修改为:

    wt.exe -w 0 new-tab -d . -p "{0caa0dad-35be-5f56-a8ff-afceeeaa6101}" "%1" %*
    1

    参数详解

    key 值 含义
    wt.exe Windows Terminal 程序
    -w 0 new-tab 在「第一个窗口/现有窗口」的「新标签页」打开
    -d . 在「文件所在目录」打开
    -p "{0caa0dad-35be..}" profile 用「指定配置」打开
    "%1" 参数列表中的第1个参数
    %* 引用所有参数

    # 「-p 配置」如何查找

    在 Windows Terminal 中找到 CMD 的 profiles Ctrl + , 打开设置 ▶ 左下角 打开 JSON 文件

    - 搜索
"name": "CMD"

- 在附近可以找到
"guid": "{0caa0dad-35be-5f56-a8ff-afceeeaa6101}"
    1
    2
    3
    4
    5

    # 2.打开 bat 文件

    新建bat文件, 写入以下内容,

    @echo off  
chcp 65001  
echo 你好  
pause

    2
    3
    4

    双击, 会在 WindowsTerminal 中打开
